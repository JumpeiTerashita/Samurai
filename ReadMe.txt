【作品】　		『最後之侍』
【使用言語】 		C#
【製作期間】 		約2ヶ月
【ゲームエンジン】	Unity
【操作方法】（Xinput コントローラー使用）

左スティック			移動
右スティック			カメラ回転（左右）
右スティック押し込み		キャラクター方向にカメラを向ける
Aボタン				ガード
攻撃をガード時 Aボタン離す	カウンター攻撃
Bボタン				攻撃
Xボタン				スキル『明鏡止水』発動

【明鏡止水システム】
画面左下の『明鏡止水』の文字（明鏡止水ゲージ）が 赤色に変化したら使用可能になります。
Xボタンを押すとカットインと共に スキル『明鏡止水』が発動します。
時間の流れを遅くし、敵の動きが緩やかになる中 プレイヤーだけが今までと同じ速度で行動できます。
（敵は遅いながらも攻撃判定は残っているので、無敵状態にはなっていません。）
明鏡止水ゲージは、敵を倒すもしくはその場で何もせずいると溜まっていきます。
