【作品】　		『最後之侍』
【使用言語】 		C#
【製作期間】 		2ヶ月
【ゲームエンジン】	Unity
【操作方法】（Xinput コントローラー使用）

左スティック			移動
右スティック			カメラ回転（左右）
右スティック押し込み		キャラクター方向にカメラを向ける
Aボタン				ガード
攻撃をガード時 Aボタン離す	カウンター攻撃
Bボタン				攻撃

【説明】
一撃死のアクションゲームで真剣勝負ならではの緊張感を表現しました。
今後、
・当たり判定の持続時間をCoroutine処理に置き換え
・敵バリエーションの追加
・新しいゲームシステムの導入
などをしていく予定です。
